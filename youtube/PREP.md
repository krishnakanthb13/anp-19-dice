### General Dice Options

#### 1. Basic Dice Roller
- **Script:** "First, let's look at the **Basic** option. This allows you to roll standard polyhedral dice.
  - **Dice Count & Face Count sub-options** let you customize how many dice and how many sides they have.
  - **Min/Max Limits sub-options** let you set threshold values.
  - **Keep/Drop sub-options** allow you to automatically discard the highest or lowest rolls.
  - **Exploding sub-option** lets dice reroll and add to the total if they land on their max value.
  - **Unique sub-option** ensures no duplicate numbers are rolled.
  - **TL;DR:** Standard dice roll with advanced filters."

#### 2. Advanced Dice Expressions
- **Script:** "Next, we have the **Advanced** option.
  - **Expression sub-option** allows you to type in complex mathematical expressions like `3d4 + 2d6 * 5`.
  - **TL;DR:** Evaluates custom mathematical dice formulas."

#### 3. Quick Roll Presets
- **Script:** "For speed, you can use the **Quick Roll Presets** option.
  - **Preset Selector sub-options** let you immediately roll a D20, 2d6, 3d6, or even run a D&D Ability Score generation.
  - **TL;DR:** Quick access to common roleplaying game presets."

#### 4. Percentile (D100)
- **Script:** "Lastly under General Dice is the **Percentile (D100)** option.
  - **Roll Range sub-option** lets you choose between 00-99 and 1-100 formats.
  - **Flip-to-Succeed sub-option** allows you to swap tens and units digits to maximize success chance.
  - **TL;DR:** Two-digit percentile rolling with optional success criteria."

---

### Game Systems Options

#### 1. Fudge/Fate
- **Script:** "Under Game Systems, we have the **Fudge/Fate** option.
  - **Dice Count sub-option** lets you decide how many custom Fudge/Fate dice to roll (default is 4).
  - **TL;DR:** Rolls standard Fate dice (`+`, ` `, `-`) and calculates the total modifier."

#### 2. Fantasy AGE Stunt - Single Roll
- **Script:** "Next is the **Fantasy AGE Stunt - Single Roll** option.
  - **Modifiers sub-option** lets you add flat bonuses to the roll.
  - **Target Number sub-option** defines the difficulty check value.
  - **TL;DR:** Simulates a single 3d6 check, tracking double results for stunt points."

#### 3. Fantasy AGE Stunt - Roll All At Once
- **Script:** "You can also run **Fantasy AGE Stunt - Roll All At Once**.
  - **Party Size sub-option** specifies how many characters roll simultaneously.
  - **TL;DR:** Executes party-wide stunt rolls simultaneously."

#### 4. Dice Pool (Shadowrun/WoD)
- **Script:** "For dice-pool systems, choose **Dice Pool (Shadowrun/WoD)**.
  - **Pool Size sub-option** sets the number of d6s to roll.
  - **Target Number sub-option** sets the minimum value needed to count as a success.
  - **Exploding sub-option** rerolls any max values.
  - **Count Type sub-option** lets you count hits or misses.
  - **TL;DR:** Accumulates successes across multiple dice."

---

### Oracles & Divination Options

#### 1. Specialized Dice
- **Script:** "For custom dice theory, use the **Specialized** option.
  - **Sicherman Dice sub-option** rolls dice with alternative face distributions (1-3-4-5-6-8 and 1-2-2-3-3-4) yielding identical sum distributions to standard d6s.
  - **Intransitive Dice sub-option** rolls three dice (A, B, C) where A beats B, B beats C, and C beats A.
  - **Poker Dice sub-option** rolls custom hands with Standard, Numeric, or Crown faces.
  - **Show Probabilities sub-option** displays theoretical mathematical win rates.
  - **TL;DR:** Simulates advanced dice designs and poker hands with optional probability calculations."

#### 2. Magic 8-Ball
- **Script:** "For quick answers, there's **8 Ball**.
  - **Question Input sub-option** takes your yes/no query.
  - **TL;DR:** Provides classic magic 8-ball answers."

#### 3. Ask Sai Baba
- **Script:** "For spiritual guidance, use **Ask Sai Baba**.
  - **Query sub-option** retrieves one of 720 wisdom responses.
  - **TL;DR:** Pulls one of 720 spiritual guidance responses."

#### 4. Tarot Cards
- **Script:** "For divination, choose **Tarot Cards**.
  - **Spread sub-option** lets you choose between a Single Card, 3-Card Spread, or Celtic Cross layout.
  - **Orientation sub-option** enables upright or reversed meanings.
  - **TL;DR:** Draws Tarot cards with descriptive meaning text."

### Generators & Tools Options

#### 1. Weighted Random
- **Script:** "Under Generators & Tools, we have **Weighted Random**.
  - **Items & Weights sub-options** let you define list items and their relative probability weights.
  - **TL;DR:** Picks a random item based on relative probability."

#### 2. Decision Matrix
- **Script:** "To resolve tough choices, use the **Decision Matrix** option.
  - **Options list sub-option** defines the choices.
  - **Criteria list sub-option** outlines what to rate them on.
  - **Weights sub-option** sets the importance of each criterion.
  - **TL;DR:** Scores options objectively using weighted criteria."

#### 3. Name Generator
- **Script:** "For characters, use the **Name Generator** option.
  - **Thematic Style sub-option** lets you choose between Fantasy, Sci-Fi, or Norse names.
  - **Gender & Titles sub-options** add extra realism.
  - **TL;DR:** Generates randomized character names with matching titles."

#### 4. Table - Randomizer (Note Action)
- **Script:** "Finally, when inside a note with a table, run the **Table - Randomizer** note option.
  - **Target Table sub-option** lets you pick which table to randomize.
  - **TL;DR:** Selects random values directly from active note tables."
